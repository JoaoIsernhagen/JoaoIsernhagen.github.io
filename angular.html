Manual de Angular

Introdução

O Angular é uma plataforma de desenvolvimento web front-end que permite a criação de aplicativos dinâmicos e interativos.

Versões

A plataforma Angular possui várias versões, sendo a mais recente a Angular 12. Certifique-se de verificar a documentação oficial de versões para obter informações sobre as versões mais recentes.

Instalando o Angular CLI

O Angular CLI (Command Line Interface) é uma ferramenta que ajuda na criação e gerenciamento de projetos Angular.

Para instalá-lo, abra um terminal e execute o seguinte comando:

npm install -g @angular/cli

Verificando a Instalação

Após a instalação, verifique se o Angular CLI foi instalado corretamente executando o comando:

ng --version

Criando um Projeto em Angular

Para criar um novo projeto Angular, navegue até a pasta desejada e execute o seguinte comando:

ng new nome-do-projeto

Executando Aplicações em Angular

Após criar o projeto, navegue até a pasta do projeto e execute o comando para iniciar um servidor de desenvolvimento:

ng serve

Isso iniciará um servidor e você poderá acessar o aplicativo no seu navegador em http://localhost:4200.
